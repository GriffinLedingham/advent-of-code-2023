# advent-of-code-2023

My problem submissions for Advent of Code 2023, built with TypeScript & Node.

https://adventofcode.com/

## Usage

- `yarn start <PROBLEM>`
  - ie. `yarn start 01_01`
 
## Test

Runs test cases w/ Jest on the problem solutions, based on the sample results provided by Advent of Code.

- `yarn test`
